# Student Portal Dashboard

A modern, clean, and responsive **Student Portal Dashboard** UI for a school web app. This project is designed for both high school and college students, using a soft blue-and-white visual style with rounded corners, subtle shadows, and smooth interactions.

## Features

- **Top Navigation Bar**
  - School logo and branding on the left.
  - Student profile menu on the right with dropdown options:
    - Settings
    - Logout

- **Sidebar Menu**
  - Dashboard
  - My Courses
  - Grades
  - Attendance
  - Assignments
  - Announcements
  - Schedule
  - Messages
  - Account Settings

- **Main Content**
  - Welcome section with a personalized greeting and motivational message.
  - Four summary cards for:
    - Current GPA
    - Total Enrolled Subjects
    - Upcoming Assignments
    - Attendance Percentage
  - Recent Grades table with Subject, Teacher, Score, and Remarks.
  - Calendar widget for upcoming exams and events.
  - Announcements section with title, date, and description.
  - Grade progress bar chart.

- **Responsive Design**
  - Optimized for desktop, tablet, and mobile.
  - Mobile sidebar toggle behavior.

- **Visual & UX Details**
  - Clean typography (Inter font).
  - Smooth hover states and light animations.
  - Soft color palette and card-based layout.

## Project Structure

- `index.html` – Dashboard structure and content
- `styles.css` – Styling, layout, responsiveness, and animations
- `script.js` – UI interactivity (profile dropdown and mobile sidebar)

## Getting Started

### 1) Clone the repository

```bash
git clone <your-repo-url>
cd AchievementsCorner
```

### 2) Open directly in your browser

You can open `index.html` directly, or run a local server (recommended).

### 3) Run a local development server

Using Python:

```bash
python3 -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/index.html
```

## Validation

You can validate JavaScript syntax with:

```bash
node --check script.js
```
